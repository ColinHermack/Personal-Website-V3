Completed over the summer of 2024 at W.R. Berkley, this is an internal tool used to monitor jobs which run on remote
servers. It is a full-stack web application with a backend written in .NET Core and the frontend in React.

## Tech Stack
C#, .NET Core, MSSQL, Octopus Deploy, TypeScript, ReactJS
